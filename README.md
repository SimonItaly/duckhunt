##### Compilation

  1.  Install the following libraries:
    - SDL 1.2
    - SDL_image 1.2
    - SDL_ttf 1.2
    - SDL_mixer 1.2
  2.  Compile the project using "compile.sh"
  3.  Run "./duck_hunt"
  4.  Have fun!
